About UniHub

UniHub is a web-based university club and event management platform. It is designed to make club communication, event organization, and student participation easier.

Current Sprint 1 Features

  • Basic frontend project setup
  • Backend PHP server setup
  • MySQL database creation
  • Database connection using PHP PDO
  • Basic API for clubs and events
  • Basic folder structure
