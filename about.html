About UniHub

UniHub is a web-based university club and event management platform. It is designed to make club communication, event organization, and student participation easier.

Current Sprint 2 Features

  • User login UI
  • Login API
  • Email and password verification
  • Session-based login
  • Profile management
  • Logout system
